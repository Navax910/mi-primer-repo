# Dashboard de Estudiante - Mecatrónica

Este proyecto es mi primera herramienta de organización personal creada durante el Módulo 1.

## Características
- **Semántica HTML5**: Uso de `header`, `main`, `section` y `footer`.
- **Enfoque Académico**: Sección de seguimiento de materias e ingeniería.
- **Formulario de Registro**: Captura de tareas con fechas y prioridades.
